# ⏰ Digital Clock & Alarm GUI

A modern and stylish **Digital Clock & Alarm Desktop Application** built with **Python**, **Tkinter**, and the built-in **time module**.

This project features a real-time digital clock, alarm management system, sound notifications, snooze functionality, blinking effects, and a beautiful dark-themed GUI.

---

# ✨ Features

- 🕒 Real-time Digital Clock
- 📅 Current Date & Day Display
- 🌤 Dynamic Greeting System
- 🔔 Multiple Alarm Support
- 💤 Snooze Functionality
- 🔊 Alarm Sound System
- 🎨 Modern Dark UI Design
- ⚡ Live Clock Updates Every Second
- ✨ Blinking Colon Animation
- ⏸ Pause & Resume Clock
- 🧪 Alarm Sound Test Button
- 🔥 Flashing Screen Effect During Alarm
- 🗑 Delete & Toggle Alarms
- 💻 Cross-Platform Support

# 🧠 How It Works

The application uses:

- `Tkinter` for creating the graphical interface
- `time module` for fetching current system time
- `threading` for smooth alarm sound playback
- `numpy` for generating WAV sound frequencies

The clock updates every second using:

```python
root.after(1000, self._tick)
```
